# Atoms

A social site with auto generated questions from OpenAI

## About this project

#### Stack

- Next.js
- tRPC
- Postgres
- OpenAI/Langchain
- Deployed on Railway

#### Features

- Sign in and answer auto generated questions from OpenAI
- Upload a new profile photo
- See others in the community
